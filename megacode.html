About Megacode

Megacode is an encoding method created by me to encode and decode messages. It is a non-deterministic code, meaning the same message encoded multiple times will produce different results.

Here's how to encode a message with Megacode:

Message: I am an encoded message 1. Shuffle your words randomly: → I an message encoded am 2. Before each word, add the position of that word in the original sentence: → 1I 3an 5message 4encoded 2am 3. Add the length of the word to the number preceding it: → 2I 5an 12message 11encoded 4am 4. Multiply each number by 3: → 6I 15an 36message 33encoded 12am 5. Apply a Caesar cipher with a shift of 1: → 6J 15bo 36nfttbhf 33fodpefe 12bn 6. Replace spaces with symbols: First space → `.`, second → `/`, third → `\`, and so on Final: 6J.15bo/36nfttbhf\33fodpefe

Special characters in the message should be replaced with a code before encoding, though in our example, none are present.
